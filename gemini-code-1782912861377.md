# 🚀 Flashcards ENEM - Memorização Inteligente

Este projeto é um repositório de flashcards focados nos conteúdos que mais caem no ENEM. O objetivo é utilizar a repetição espaçada para fixar fórmulas, conceitos e regras essenciais de forma rápida e prática.

## 🗂️ Organização das Pastas

O repositório está dividido por áreas do conhecimento:

- `/ciencias-da-natureza`: Fórmulas de Física, Química e conceitos de Biologia.
- `/matematica`: Geometria, Álgebra, Estatística e Trigonometria.
- `/humanas`: Linhas do tempo de História, conceitos de Sociologia e Filosofia.
- `/linguagens`: Figuras de linguagem e regras gramaticais.

## 💡 Como Usar estes Flashcards

Você pode utilizar estes cartões de duas formas:

1. **Importando no Anki:** Baixe os arquivos `.apkg` (se disponíveis) ou importe os arquivos `.csv` direto no aplicativo Anki.
2. **Revisão Direta no GitHub:** Abra os arquivos markdown (`.md`) de cada matéria. A pergunta estará visível e a resposta estará escondida sob a tag de spoiler.

### Exemplo de Flashcard (Markdown):
<details>
  <summary><b>Pergunta:</b> Qual é a fórmula da Primeira Lei de Ohm?</summary>
  <br>
  <b>Resposta:</b> $U = R \cdot i$ (onde U é diferença de potencial, R é resistência e i é a corrente elétrica).
</details>

## 🛠️ Como Contribuir

1. Faça um **Fork** do projeto.
2. Crie uma nova branch com seus flashcards: `git checkout -b meus-flashcards`.
3. Adicione suas perguntas e respostas seguindo o padrão de organização.
4. Abra um **Pull Request**.
